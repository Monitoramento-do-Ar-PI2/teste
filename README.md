# Monitoramento do Ar

### Para contribuir:


1. Instalação pip
   ```
    pip install --upgrade pip
   ```

2. Instalação o MkDocs
   ```
   pip install mkdocs
   ```

3. Clonar o repositório
    ```
    git clone https://github.com/Monitoramento-do-Ar-PI2/wiki.git
    ```

4. Entre na pasta do projeto
   ```
   cd wiki
   ```

5. Rode a wiki
   ```
   mkdocs serve
   ```
